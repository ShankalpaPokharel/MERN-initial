Concept of pre tag

The <pre> tag in HTML is used to define preformatted text. When you place content within a <pre> element, it is displayed in a fixed-width font, and the text preserves both spaces and line breaks. Essentially, the text inside a <pre> element is displayed exactly as written in the HTML source code.
